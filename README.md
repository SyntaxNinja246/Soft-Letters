# Soft-Letters
Soft Letters is an interactive journaling web application designed to help users express emotions through writing. By selecting a mood, users can create personal letters and track their emotional patterns using a dynamic mood map that visualizes entries for each feeling.

## 🌸 About the Project
Soft Letters is a mood-based journaling web app where users can write letters based on how they feel.  
Each emotion is mapped, helping users visualize their emotional journey over time.

## 🎭 Features
- ✍️ Write letters based on your mood
- 😊 Mood selection (happy, sad, anxious, etc.)
- 🗺️ Mood Map showing number of letters per emotion
- 📊 Emotional insights (track your feelings)
- 💾 Save and revisit your thoughts

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript
